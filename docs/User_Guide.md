# User Guide

## Students

- Register via student/register.php
- Login via student/login.php
- Dashboard: view details, attendance %, active count, and downloads
- Download files via student/download.php?id=...
- Logout via student/logout.php

## Admins

- Login via admin/login.php
- Dashboard: view metrics
- Attendance: mark and view attendance
- Uploads: add notes/photos securely
- Notices: publish announcements
- Enquiries: view and filter contact messages
- Appointments: view and filter bookings
- Logout via admin/logout.php
